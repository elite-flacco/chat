## [0.1.0] - 2025-09-08

### Added

- GitHub workflows for continuous integration
- Claude Code commands and project configuration
- AI chatbot application with Next.js
- Support for multiple AI providers (OpenAI and Anthropic)
- Model selection interface (GPT-4o, GPT-4o Mini, GPT-5, GPT-5 mini, Claude 4 Sonnet)
- Web search tool integration for OpenAI models
- TypeScript support with strict mode
- ESLint and Prettier configuration
- Pre-commit hooks via Husky
- Jest testing setup with React Testing Library
- TailwindCSS styling
